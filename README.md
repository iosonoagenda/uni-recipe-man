# uni-recipe-man

## Project setup

```
yarn install
```

### Compiles and run for development

```
yarn start
```

### Compiles and minifies for production

```
yarn run dist
```

```
yarn run dist:win
```

```
yarn run dist:lin
```

```
yarn run dist:mac
```
